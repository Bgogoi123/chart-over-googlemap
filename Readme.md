# 🌏 Geographical Visualization of Diseases

> Representation of Disease and related information in Circular Charts on top of a Geographical Map (Specifically, Google Map). 


### 🤖 Features:
- ✅ Select Single or Multiple Diseases.
- 📅 View Data for specific Dates.
- 🔍 Zoom in/out, Panning, and Full Screen View on the Google map.
- ☝🏽 View Disease Count and relative data on clicking a Circular Chart.

## 🛠 Tech Stack

- React + TypeScript
- Material UI
- Google Maps API
